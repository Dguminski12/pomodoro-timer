# Pomodoro Timer

A dependency-free 25-minute focus timer built with semantic HTML, CSS, and vanilla JavaScript.

## Features

- Start, pause, and reset controls
- Guard against creating multiple timer intervals
- Zero-padded `MM:SS` display
- Automatic stop at `00:00`
- No build step or external runtime dependencies

## Run locally

```bash
git clone https://github.com/Dguminski12/pomodoro-timer.git
cd pomodoro-timer
```

Open `index.html` in a browser.

## Possible next steps
